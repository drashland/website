Features

Even though it's micro, Drash comes with some power.

Resources

Create resources, define their URIs, and allow clients to make requests to them.

Content Negotiation

Send clients the representations they request — JSON, XML, HTML, etc.

Middleware

Filter requests globally, or be more granular with resource-level middleware.

Request Body Parsing

Parse JSON, URL encoded, and multipart bodies with a single method call.

Route Path Params

Support your REST API endpoints with route path params.

Serve Static Assets

Serve static assets like stylesheets and JavaScript files — good for web apps.
